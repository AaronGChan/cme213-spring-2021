# To recreate the web site on your local computer

[Install Jeklly with Bundler](https://jekyllrb.com/tutorials/using-jekyll-with-bundler/).

Then:

```
$ bundle install
```

To generate the site:

```
$ bundle exec jekyll serve --port=4000 --incremental
```
